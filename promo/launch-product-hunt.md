## Description
```
My Hangul Name converts any name, from any language, into its natural Korean (Hangul) spelling —
based on real pronunciation and Korea's official transcription rules. Free, instant, with IPA,
audio, 20+ fonts, and shareable Hangul art.

For developers: a free no-key API (CORS-enabled) and a hosted MCP server, so Claude and other
agents can convert names natively.
```
